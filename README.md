# RtosExPro
This is a repository for rtos sample project.

本仓库为公众号"不解之榬"的博文《RTOS》系列的配套示例仓库，其主要功用有：
1. freertos分支为freertos系列博文的配套示例代码
2. rtt分支为RT-Thread系列博文的配套示例代码

#### 特别说明：
主分支中有博文中提到的KEIL编译、烧写脚本
